## 📖 About

SignatureStudio is a modern web-based digital signature application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

It provides an intuitive drawing experience with a clean, responsive interface, allowing users to create, customize, save, retrieve, and download digital signatures directly from their browser. Lightweight and dependency-free, it's designed to be fast, responsive, and easy to use across desktop and mobile devices.

---

## ✨ Features

* ✍️ Smooth Signature Drawing
* 🎨 Brush Color Picker
* 🖼️ Canvas Background Color Picker
* 📏 Adjustable Brush Size
* 🎯 Live Brush Cursor
* 💾 Auto Save with Local Storage
* 📂 Retrieve Saved Signature
* ⬇️ Download Signature as PNG
* ↩️ Undo Support
* 📱 Touch & Mouse Support
* 🌙 Modern Glassmorphism UI
* ⚡ Fully Responsive Design
* 🔔 Toast Notifications

## ⌨️ Keyboard Shortcuts

| Shortcut     | Action             |
| ------------ | ------------------ |
| **Ctrl + Z** | Undo               |
| **Ctrl + S** | Download Signature |
| **Delete**   | Clear Canvas       |

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/KaustubhDeshmane/SignatureStudio.git
```

### Navigate to the project

```bash
cd SignatureStudio
```

### Run the project

Open **index.html** in your preferred web browser.

No installation, build tools, or external dependencies are required.
